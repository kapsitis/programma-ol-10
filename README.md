# programma-ol-10

Programmas paraugs Matemātika 1 (optimālā līmeņa) 10.klasei.
Projekts satur kursa "Matemātika 1" jeb vidusskolas 
matemātikas optimālā līmeņa (OL) programmas paraugu 10.klasei. 
Tajā ir arī ģeneratīvajiem valodu modeļiem paredzēti 
skripti un uzvednes jeb prompti, lai analizētu 
failus šajā vidusskolas programmas paraugā.

Analīzes gaitā noskaidrojam programmas 
atbilstību spēkā esošajiem 
standartiem, teorijas materiālu un uzdevumu (arī darbalapu) 
piemērotību un izveidojam dažus pārskatus, lai 
saprastu optimālā līmeņa matemātikas 
kursa tvērumu un programmas paraugā esošo failu 
atbilstību tam.

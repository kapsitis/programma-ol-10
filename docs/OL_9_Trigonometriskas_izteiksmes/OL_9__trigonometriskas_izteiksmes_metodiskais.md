**1.--4. stunda. Trigonometriskās pamatidentitātes**

**Sasniedzamie rezultāti**

- Lieto trigonometriskās pamatidentitātes (un secinājumus no tām)
  $\sin^{2}{\alpha\  + \cos^{2}{\alpha = 1}}$,\
  $tg\alpha = \frac{\sin\alpha}{\cos\alpha}$,
  $ctg\alpha = \frac{\cos\alpha}{\sin\alpha}$ un
  $tg\ \alpha \cdot ctg\ \alpha = 1$ izteiksmju pārveidojumos.

- Aprēķina trigonometrisko izteiksmju skaitliskās vērtības.

*Temata sākumā atkārto un uzsver svarīgāko par trigonometriskajām
funkcijām.*

Trigonometriskajā vienības riņķī šaurā leņķa sinuss ir vienāds ar punkta
$P(x;y)$ ordinātu jeb ${sin\ \alpha}{= y}$, kosinuss -- ar punkta $P$
abscisu jeb $\cos{\alpha = x}$, tangenss -- ar ordinātas un abscisas
dalījumu jeb $tg\ \alpha = \frac{y}{x}$, kotangenss -- ar abscisas un
ordinātas dalījumu jeb $ctg\ \alpha = \frac{x}{y}$.

Vienkāršojot trigonometrisku izteiksmi, tiek izmantotas gan
trigonometriskās sakarības (argumenti var mainīties), gan algebriskie
pārveidojumi (argumenti nemainās).

Veicot algebriskos pārveidojumus ar viena argumenta trigonometriskajām
funkcijām arguments nemainās, piemēram, $\sin x + \sin x = 2\sin x$.

Veic trigonometrisko izteiksmju algebriskus pārveidojumus (skat.
[OL_9_dl_1]{.mark} 1. uzdevumu).

**Sakarības starp viena un tā paša argumenta trigonometriskajām
funkcijām**

Aplūko trigonometrisko vienības riņķi.

Ikviena vienības riņķa līnijas punkta $P(x;y)$ koordinātas apmierina
vienādojumu

$$x^{2} + y^{2} = 1.$$

Lietojot definīcijas $y = \sin\alpha$ un $x = \cos\alpha$, iegūst, ka
visām leņķa $\alpha$ vērtībām izpildās vienādība

$$\sin^{2}{\alpha + \cos^{2}\alpha = 1.}$$

Identitātes $\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$ kreisās puses
izteiksmi $\sin^{2}{\alpha + \cos^{2}\alpha}$ sauc arī par
trigonometrisko vieninieku.

Vienkāršo!

  ----------------------------------------------------------------------------------------------
  **1.**   $$\sin^{2}{2\alpha + \cos^{2}{2\alpha} = 1}$$
  -------- -------------------------------------------------------------------------------------
  **2.**   $$\sin^{2}{2\alpha + \cos^{2}{2\alpha} = 1}$$

  **3.**   $${\sin^{2}5}{\alpha + {\cos^{2}5}\alpha = 1}$$

  **4.**   $$\sin^{2}{\beta + \cos^{2}\beta = 1}$$

  **5.**   $$3\sin^{2}x + 3\cos^{2}x = 3\left( \sin^{2}x + \cos^{2}x \right) = 3 \cdot 1 = 3$$

  **6.**   $$- 2\sin^{2}x - 2\cos^{2}x = - 2\left( \sin^{2}x + \cos^{2}x \right) = - 2$$
  ----------------------------------------------------------------------------------------------

*Ar ko vienāda izteiksme* $\sin^{2}\alpha$*?*

$$\sin^{2}\alpha = 1 - \cos^{2}\alpha$$

*Ar ko vienāda izteiksme* $\cos^{2}\alpha$*?*

$$\cos^{2}\alpha = 1 - \sin^{2}\alpha$$

**!!!** Formulas ar tangensu un kotangensu nav optimālā līmeņa saturs,
tāpēc tās var nemācīt un atbilstošos uzdevumus neapskatīt.

*Kādas sakarības var iegūt, lietojot tangensa un kotangensa definīciju,
ja punkts\*
$P(x;y) = P({\cos\ \alpha}{;\sin\alpha})$ *ir trigonometriskās vienības
riņķa līnijas punkts?*

Pēc definīcijas:

$$tg\ \alpha = \frac{y}{x} = \frac{\sin\alpha}{\cos\alpha};$$

$$ctg\ \alpha = \frac{x}{y} = \frac{\cos\alpha}{\sin\alpha}.$$

Uzraksti kā trigonometrisko funkciju dalījumu!

  --------------------------------------------------------------
  **1.**   $$tg\ \beta = \frac{\sin\beta}{\cos\beta}$$
  -------- -----------------------------------------------------
  **2.**   $$tg\ 3x = \frac{\sin{3x}}{\cos{3x}}$$

  **3.**   $$ctg\ 4\beta = \frac{\cos{4\beta}}{\sin{4\beta}}$$
  --------------------------------------------------------------

*Ko var secināt par viena un tā paša argumenta tangensa un kotangensa
reizinājumu?*

Reizinot $tg\ \alpha = \frac{y}{x} = \frac{\sin\alpha}{\cos\alpha}$ un
$ctg\ \alpha = \frac{x}{y} = \frac{\cos\alpha}{\sin\alpha}$, iegūst

$$tg\ \alpha \bullet ctg\ \alpha = \frac{\sin\alpha}{\cos\alpha} \bullet \frac{\cos\alpha}{\sin\alpha} = 1.$$

Tātad viena un tā paša argumenta tangensa un kotangensa reizinājums ir
1:

$$tg\ \alpha \bullet ctg\ \alpha = 1.$$

Vienādība ir spēkā visiem $\alpha$, kuriem tangenss un kotangenss ir
definēts, t. i., visiem $\alpha$, kuriem $\sin{\alpha \neq 0}$ un
$\cos{\alpha \neq 0}$, t. i., $\alpha \neq 90{^\circ} \cdot n$, kur
$n \in Z$.

Izteiksmju $tg\ \alpha$ un $ctg\ \alpha$ ir savstarpēji apgriezti
lielumi:

$tg\alpha = \frac{1}{ctg\alpha}$ un $ctg\alpha = \frac{1}{tg\alpha}$.

Pārveido!

  -----------------------------------------------------------------
  **1.**   $$tg\ 3\alpha \bullet ctg\ 3\alpha = 1$$
  -------- --------------------------------------------------------
  **2.**   $$tg\ 3\alpha = \frac{1}{ctg\ 3\alpha}$$

  **3.**   $$\frac{1}{tg\ 7\alpha} = ctg\ 7\alpha$$

  **4.**   $$tg\frac{\alpha}{2} \bullet ctg\frac{\alpha}{2} = 1$$

  **5.**   $$tg\ \beta = \frac{1}{ctg\ \beta}$$

  **6.**   $$ctg\frac{\alpha}{5} = \frac{1}{tg\frac{\alpha}{5}}$$
  -----------------------------------------------------------------

*Aplūko, kādas vēl sakarības var iegūt no trigonometriskā vieninieka.*

Dalot identitātes $\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$ abas puses
ar $\sin^{2}{\alpha \neq 0}$ (t. i., $\alpha \neq 180{^\circ} \cdot n$,
kur $n\mathbb{\in Z}$), iegūst:

$$\frac{\sin^{2}\alpha}{\sin^{2}\alpha} + \frac{\cos^{2}\alpha}{\sin^{2}\alpha} = \frac{1}{\sin^{2}\alpha};$$

$$1 + ctg^{2}\alpha = \frac{1}{\sin^{2}\alpha}.$$

Dalot identitātes ${\sin^{2}\alpha}{+ \cos^{2}\alpha = 1}$ abas puses ar
$\cos^{2}{\alpha \neq 0}$
($\alpha \neq 90{^\circ} + 180{^\circ} \bullet n$, kur
$n\mathbb{\in Z}$), iegūst:

$$\frac{\sin^{2}\alpha}{\cos^{2}\alpha} + \frac{\cos^{2}\alpha}{\cos^{2}\alpha} = \frac{1}{\cos^{2}\alpha};$$

$$1 + tg^{2}\alpha = \frac{1}{\cos^{2}\alpha}.$$

*Šo sakarību iegūšanu var apskatīt kā uzdevumu.*

Vienkāršo izteiksmi!

  ------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **1.**   $$1 + tg^{2}\alpha = 1 + \frac{\sin^{2}\alpha}{\cos^{2}\alpha} = \frac{\cos^{2}{\alpha + \sin^{2}\alpha}}{\cos^{2}\alpha} = \frac{1}{\cos^{2}\alpha}$$
  -------- ---------------------------------------------------------------------------------------------------------------------------------------------------------
  **2.**   $$1 + ctg^{2}\alpha = 1 + \frac{\cos^{2}\alpha}{\sin^{2}\alpha} = \frac{\sin^{2}{\alpha + \cos^{2}\alpha}}{\sin^{2}\alpha} = \frac{1}{\sin^{2}\alpha}$$

  ------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Trigonometriskās identitātes**

Par trigonometrisko identitāti sauc tādu vienādību, kas satur viena
argumenta vai vairāku argumentu trigonometriskās funkcijas un ir patiesa
visām šo argumentu pieļaujamām vērtībām.

Piemēri

**1.** Vienādība $\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$ ir
trigonometriska identitāte; tā ir patiesa jebkurai leņķa $\alpha$
vērtībai.

**2.** Vienādība $tg\ \alpha = \frac{\sin\alpha}{\cos\alpha}$ ir
trigonometriska identitāte, jo tā ir patiesa visiem leņķiem $\alpha$,
izņemot tos, kuriem tangenss neeksistē.

**3.** Vienādība $tg\ \alpha \bullet ctg\ \alpha = 1$ ir identitāte, jo
tā ir patiesa visiem leņķiem $\alpha$, izņemot tos, kuriem tangenss vai
kotangenss neeksistē.

> Tā kā $tg\alpha = \frac{\sin\alpha}{\cos\alpha}$, tad $tg\ \alpha$
> neeksistē, ja $\cos{\alpha = 0}$, t. i., ja
> $\alpha = 90{^\circ} + 180{^\circ} \bullet n$, kur $n \in Z$.
>
> Tā kā $ctg\ \alpha = \frac{\cos\alpha}{\sin\alpha}$, tad $ctg\ \alpha$
> neeksistē, ja $\sin{\alpha = 0}$, t. i., ja
> $\alpha = 180{^\circ} \bullet n$, kur $n \in Z$.

Lai pierādītu identitāti, var lietot divas dažādas pieejas.

1\. Izvēlas identitātes vienu pusi (kreiso vai labo) un, lietojot
sakarības starp trigonometriskajām funkcijām, aizstāj vienas funkcijas
sakarības ar citām -- identiski vienādām. Ja rezultātā iegūst tādu pašu
izteiksmi, kāda dota identitātes otrā pusē, tad identitāte ir pierādīta.

2\. Ja identitātes abās pusēs ir samērā sarežģītas izteiksmes, tad
pārveido abas identitātes puses, aizstājot vienu izteiksmi (vai vairākas
izteiksmes) ar citām, identiski vienādām izteiksmēm. To dara, līdz abās
pusēs iegūst vienādas izteiksmes. No tā secina dotās identitātes
patiesumu.

**1.** Pierādi identitāti
$\sin^{2}{\alpha \bullet ctg^{2}\alpha + \sin^{2}{\alpha = 1}}$.

Pārveido vienādības kreisās puses izteiksmi, kamēr iegūst labās puses
izteiksmi:

$$\sin^{2}\alpha \bullet ctg^{2}\alpha + \sin^{2}\alpha = \sin^{2}\alpha \bullet \frac{\cos^{2}\alpha}{\sin^{2}\alpha} + \sin^{2}\alpha = \cos^{2}\alpha + \sin^{2}\alpha = 1.$$

Tā kā vienādības kreisā puse ir vienāda ar labo pusi, tad identitāte ir
pierādīta.

**2.** Pierādi identitāti $\sin\alpha \bullet ctg\ \alpha = \cos\alpha$.

Pārveido vienādības kreiso pusi, kamēr iegūst, ka abas puses ir
vienādas:

$$\sin\alpha \bullet ctg\ \alpha\overset{?}{=}\cos{\alpha;}$$

$$\sin\alpha \bullet \frac{\cos\alpha}{\sin\alpha}\overset{?}{=}\cos{\alpha;}$$

$$\cos\alpha = \cos{\alpha.}$$

Tā kā vienādības kreisā puse ir vienāda ar labo pusi, tad identitāte ir
pierādīta.

Lieto trigonometriskās pamatidentitātes izteiksmju vienkāršošanai (skat.
[OL_9_dl_1]{.mark} 2. uzdevumu).

Aprēķini $\sin\alpha$, $tg\ \alpha$ un $ctg\ \alpha$ vērtību, ja
$\cos\alpha = - 0,8$ un $\alpha$ atrodas trešajā kvadrantā!

1\. veids. No formulas $\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$ izsaka
$\sin^{2}{\alpha = 1 - \cos^{2}\alpha}$.

Tātad $\sin^{2}\alpha = 1 - ( - 0,8)^{2} = 1 - 0,64 = 0,36$ un
$\sin\alpha = \pm \sqrt{0,36} = \pm 0,6$.

negatīva, tad $\sin\alpha = - 0,6$.

  ------------------------------------------------------------------------------------- -----------------------------------------------------

**5.--7. stunda. Redukcijas formulas**

**Sasniedzamie rezultāti**

- Pamato sakarības starp vienādu un dažādu argumentu sinusiem un
  kosinusiem (redukcijas formulas), lietojot vienības riņķi.

- Aprēķina trigonometrisko izteiksmju vērtības, lietojot redukcijas
  formulas.

- Nosaka dotajai izteiksmei identisku izteiksmi, lietojot, piemēram,
  redukciju vai līdzīgu saskaitāmo savilkšanu.

*Lai aprēķinātu dažādu leņķu trigonometrisko funkciju vērtības,
ieteicams no galvas zināt trigonometrisko funkciju vērtības*
$30{^\circ}$*;* $45{^\circ}$*;*$60{^\circ}$ *leņķiem.*

*Pēc skolotāja ieskatiem un skolēnu vajadzībām var izvēlēties, vai
leņķus ņemt radiānos vai grādos.*

*Materiāla teorētiskajā izklāstā pārsvarā lietoti radiāni, jo šādā
gadījumā nerodas matemātiska neprecizitāte ar funkcijas periodu, kas ir
pozitīvs skaitlis (nevis lielums grādos). Skolotājs pēc saviem ieskatiem
un skolēnu vajadzībām var izvēlēties vairāk lietot grādus.*

Aplūkosim, kā trigonometriskās funkcijas, kuru leņķis ir formā
$\frac{\pi}{2}n \pm \alpha$, kur $n\mathbb{\in Z}$, var izteikt ar leņķa
$\alpha$ funkcijām.

*Kā pārveidot izteiksmes* $\sin{(4\pi + \alpha)}$*,*
$\cos{(15\pi + \alpha})$*,* $tg(7\pi - \alpha)$*?*

> Var lietot faktu, ka trigonometriskās funkcijas ir periodiskas. Sinusa
> un kosinusa funkcijām periods ir $2\pi$, bet tangensa un kotangensa
> funkcijām periods ir $\pi$. Tātad izteiksmes var pārveidot šādi:

- $\sin{(4\pi + \alpha)} = \sin{(4\pi + \alpha - 2 \cdot 2\pi)} = \sin\alpha$;

- $\cos{(15\pi + \alpha}) = \cos{(15\pi + \alpha - 7 \cdot 2\pi)} = \cos{(\pi + \alpha})$;

- $tg(7\pi - \alpha) = tg(7\pi - \alpha - 7 \cdot \pi) = tg( - \alpha) = - tg\ \alpha$.

Ja argumenta vērtība nav intervālā $\lbrack 0;2\pi\rbrack$, argumentam
pieskaita vai no tā atņem šīs trigonometriskās funkcijas periodu $T$
veselu skaitu reižu tā, lai iegūtais arguments būtu robežās\
no $0$ līdz $2\pi$.

Jāievēro, ka $\sin{(\alpha + 2\pi \bullet n) = \sin\alpha}$ un
$\cos{(\alpha + 2\pi \bullet n) = \cos\alpha}$, kur $n\mathbb{\in Z}$,
jo šo funkciju periods $T = 2\pi$.

Savukārt $tg(\alpha + \pi \bullet n) = tg\alpha$ un
$ctg(\alpha + \pi \bullet n) = ctg\alpha$, kur $n\mathbb{\in Z}$, jo šo
funkciju periods $T = \pi$.

Ja leņķis ir intervālā $\lbrack 0;2\pi\rbrack$, tad to var uzrakstīt
kādā no formām $\frac{\pi}{2} \pm \alpha$; $\pi \pm \alpha$;
$\frac{3\pi}{2} \pm \alpha$; $2\pi - \alpha$, kur $\alpha$ ir šaurs
leņķis.

Izsaki leņķi kādā no formām $\frac{\pi}{2} \pm \alpha$;
$\pi \pm \alpha$; $\frac{3\pi}{2} \pm \alpha$; $2\pi - \alpha$!

  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **1.**   $$\frac{2\pi}{3} = \pi - \frac{\pi}{3}$$     vai      $$\frac{2\pi}{3} = \frac{4\pi}{6} = \frac{3\pi}{6} + \frac{\pi}{6} = \frac{2\pi}{3} = \frac{\pi}{2} + \frac{\pi}{6}$$
  -------- -------------------------------------------- -------- -----------------------------------------------------------------------------------------------------------------------
           $$120{^\circ} = 180{^\circ} - 60{^\circ}$$   vai      $$120{^\circ} = 90{^\circ} + 30{^\circ}$$

  **2.**   $$\frac{7\pi}{6} = \pi + \frac{\pi}{6}$$              $$\frac{7\pi}{6} = \frac{9\pi}{6} - \frac{2\pi}{6} = \frac{3\pi}{2} - \frac{\pi}{3}$$

           $$210{^\circ} = 180{^\circ} + 30{^\circ}$$   vai      $$220{^\circ} = 270{^\circ} - 50{^\circ}$$

  **3.**   $$\frac{7\pi}{4} = 2\pi - \frac{\pi}{4}$$    vai      $$\frac{7\pi}{4} = \frac{6\pi}{4} + \frac{\pi}{4} = \frac{3\pi}{2} + \frac{\pi}{4}$$

           $$315{^\circ} = 360{^\circ} - 45{^\circ}$$   vai      $$315{^\circ} = 270{^\circ} + 45{^\circ}$$
  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Formulas, ar kuru palīdzību leņķu $\frac{\pi}{2} \pm \alpha$;
$\pi \pm \alpha$; $\frac{3\pi}{2} \pm \alpha$; $2\pi - \alpha$
trigonometriskās funkcijas izsaka ar šaurā leņķa $\alpha$
trigonometriskajām funkcijām, sauc par redukcijas formulām.

*Principā ar redukcijas formulām tiek vienkāršota trigonometriskā
izteiksme.*

Aplūko, kā iegūt redukcijas formulas -- galvenā ideja ir trigonometrisko
funkciju definīciju un trigonometriskā vienības riņķa (un simetrijas
tajā) lietošana.

*Atkārto trigonometrisko funkciju definīcijas.*

Vienības riņķa līnijas punkta $P(a;b)$:

- abscisa $a$ ir pagrieziena leņķa $\alpha$ kosinuss,

- ordināta $b$ ir pagrieziena leņķa $\alpha$ sinuss,

- attiecība $\frac{b}{a}$ ir pagrieziena leņķa $\alpha$ tangenss,

- attiecība $\frac{a}{b}$ ir pagrieziena leņķa $\alpha$ kotangenss.

Pieņemsim, ka dota vienības riņķa līnija un uz tās punkts $P_{0}(1;0)$.

Pagriežot punktu $P_{0}(1;0)$ ap koordinātu sākumpunktu pozitīvā
virzienā par šauru leņķi $\alpha$, iegūst punktu $P_{\alpha}(a;b)$.

Salīdzinot punkta $P_{2}( - a;b)$ ar punkta $P_{\alpha}(a;b)$
koordinātām, iegūstam sakarības:

$$\cos{(\pi - \alpha) = - a = - \cos\alpha};$$

$$\sin{(\pi - \alpha) = b = \sin\alpha};$$

$$tg(\pi - \alpha) = \frac{b}{- a} = - tg\ \alpha;$$

$$ctg(\pi - \alpha) = \frac{- a}{b} = - ctg\ \alpha.$$

Var aplūkot leņķu $\frac{\pi}{2} \pm \alpha$; $\pi \pm \alpha$;
$\frac{3\pi}{2} \pm \alpha$, $2\pi - \alpha$ redukcijas formulu iegūšanu
(skat. [OL_9_dl_2]{.mark} papildu lapu, *pēc kuras aizpildīšanas var
uzdot skolēniem izdomāt veidu, kā viegli atcerēties visas formulas,
ievērojot, ka dažreiz mainās zīme un dažreiz funkcijas nosaukums*), vai
arī vienkārši formulēt reducēšanas likumus.

Lai vieglāk atcerēties reducētās funkcijas nosaukuma noteikšanu, var
ievērot, kāda ir galvas kustība, atbildot uz jautājumu "Vai nosaukums
mainās?", sakot "Jā", un kāda -- sakot "Nē".

*Uzsvērt, ka šis nav formulu pierādījums, bet gan tikai veids, kā
formulas atcerēties.*

Redukcijas formulu lietošanu var arī formulēt kā likumu:

1\. likums (reducētās funkcijas zīmes noteikšanai) -- funkcijai, ar kuru
aizstāj, ir tāda pati zīme kā dotajai funkcijai atbilstošajā kvadrantā;

2\. likums (reducētās funkcijas nosaukuma noteikšanai):

- ja dotās funkcijas arguments (leņķis) ir $\pi \pm \alpha$ vai
  $2\pi \pm \alpha$, tad funkcijas nosaukums nemainās;

- ja dotās funkcijas arguments (leņķis) ir $\frac{\pi}{2} \pm \alpha$
  vai $\frac{3\pi}{2} \pm \alpha$, tad funkcijas nosaukums mainās šādi:
  $sin \longleftrightarrow cos$ un $tg \longleftrightarrow ctg$.

**Sasniedzamie rezultāti**

- Lieto zināšanas vai lieto pretpiemēru, lai atspēkotu aplamus

- Veido pierādījumu vai skaidro dotu pierādījumu argumentu saskaitīšanas

*Kā aprēķināt izteiksmes* $\sin{(30{^\circ} + 45{^\circ})}$ *vērtību?*

> $\sin{75{^\circ}} \approx 0,9659$*.*

$\sin{30{^\circ}} + \sin{45{^\circ}} = \frac{1}{2} + \frac{\sqrt{2}}{2} = \frac{1 + \sqrt{2}}{2}$*?*

argumentu summas formulai dažām leņķu* $\alpha$ *un* $\beta$ *vērtībām

$$\sin(\alpha + \beta) = \sin\alpha \bullet \cos\beta + \cos\alpha \bullet \sin\beta;$$

**Argumentu saskaitīšanas formulas**

Formulas, kas divu leņķu summas vai starpības funkcijas izsaka ar šo
leņķu funkcijām, sauc par argumentu saskaitīšanas formulām.

Divu leņķu summas sinuss vienāds ar summu, kurā viens saskaitāmais ir
pirmā leņķa sinusa reizinājums ar otrā leņķa kosinusu, bet otrs
saskaitāmais ir pirmā leņķa kosinusa reizinājums ar otrā leņķa sinusu.

$$\mathbf{\sin}\left( \mathbf{\alpha}\mathbf{+}\mathbf{\beta} \right)\mathbf{=}\mathbf{\sin}\mathbf{\alpha \bullet}\mathbf{\cos}\mathbf{\beta +}\mathbf{\cos}\mathbf{\alpha \bullet}\mathbf{\sin}\mathbf{\beta}$$

Divu leņķu summas kosinuss ir vienāds ar šo leņķu kosinusu reizinājuma
un šo pašu leņķu sinusu reizinājuma starpību.

$$\mathbf{\cos}\left( \mathbf{\alpha}\mathbf{+}\mathbf{\beta} \right)\mathbf{=}\mathbf{\cos}\mathbf{\alpha \bullet}\mathbf{\cos}\mathbf{\beta -}\mathbf{\sin}\mathbf{\alpha \bullet}\mathbf{\sin}\mathbf{\beta}$$

*Kā iegūt formulas* $\sin{(\alpha - \beta)}$ *un*
$\cos{(\alpha - \beta})$*?*

Lieto sakarību, ka $\alpha - \beta = \alpha + ( - \beta)$:

$$\sin{(\alpha - \beta)} = \sin\left( \alpha + ( - \beta) \right) = \sin\alpha \bullet \cos( - \beta) + \cos\alpha \bullet \sin( - \beta) =$$

$$= \sin\alpha \bullet \cos\beta - \cos\alpha \bullet \sin\beta;$$

$$\cos(\alpha - \beta) = \cos\left( \alpha + ( - \beta) \right) = \cos\alpha \bullet \cos( - \beta) + \sin\alpha \bullet \sin( - \beta) =$$

$$= \cos\alpha \bullet \cos\beta + \sin\alpha \bullet \sin\beta.$$

Divu leņķu starpības sinuss ir pirmā leņķa sinusa reizinājums ar otrā
leņķa kosinusu mīnus pirmā leņķa kosinusa reizinājums ar otrā leņķa
sinusu.

$$\mathbf{\sin}\left( \mathbf{\alpha}\mathbf{-}\mathbf{\beta} \right)\mathbf{=}\mathbf{\sin}\mathbf{\alpha \bullet}\mathbf{\cos}\mathbf{\beta -}\mathbf{\cos}\mathbf{\alpha \bullet}\mathbf{\sin}\mathbf{\beta}$$

Divu leņķu starpības kosinuss ir vienāds ar šo leņķu kosinusu
reizinājuma un šo pašu leņķu sinusu reizinājuma summu.

$$\mathbf{\cos}\left( \mathbf{\alpha}\mathbf{-}\mathbf{\beta} \right)\mathbf{=}\mathbf{\cos}\mathbf{\alpha \bullet}\mathbf{\cos}\mathbf{\beta +}\mathbf{\sin}\mathbf{\alpha \bullet}\mathbf{\sin}\mathbf{\beta}$$

Lieto argumentu saskaitīšanas formulas, lai divu leņķu summas tangensu
izteiktu ar šo leņķu tangensiem:

$$tg(\alpha + \beta) = \frac{\sin{(\alpha + \beta)}}{\cos{(\alpha + \beta)}} = \frac{\sin{\alpha \bullet \cos{\beta + \cos{\alpha \bullet \sin\beta}}}}{\cos{\alpha \bullet \cos{\beta - \sin{\alpha \bullet \sin\beta}}}}.$$

Izteiksmē skaitītāju un saucēju izdala ar
$\cos{\alpha \bullet \cos\beta}$ un izsaka ar $tg\ \alpha$ un
$tg\ \beta$:

$$tg(\alpha + \beta) = \frac{\frac{\sin{\alpha \bullet \cos{\beta + \cos{\alpha \bullet \sin\beta}}}}{\cos{\alpha \bullet \cos\beta}}}{\frac{\cos{\alpha \bullet \cos{\beta - \sin{\alpha \bullet \sin\beta}}}}{\cos{\alpha \bullet \cos\beta}}} = \frac{\frac{\sin{\alpha \bullet \cos\beta}}{\cos{\alpha \bullet \cos\beta}} + \frac{\cos{\alpha \bullet \sin\beta}}{\cos{\alpha \bullet \cos\beta}}}{\frac{\cos{\alpha \bullet \cos\beta}}{\cos{\alpha \bullet \cos\beta}} - \frac{\sin{\alpha \bullet \sin\beta}}{\cos{\alpha \bullet \cos\beta}}} = \frac{tg\alpha + tg\beta}{1 - tg\alpha \bullet tg\beta}.$$

Ja $\cos{(\alpha + \beta)} \neq 0$, $\cos{\alpha \neq 0}$ un
$\cos{\beta \neq 0}$, tad ir spēkā formula

$$\mathbf{tg}\left( \mathbf{\alpha + \beta} \right)\mathbf{=}\frac{\mathbf{tg\ }\mathbf{\alpha +}\mathbf{tg\ }\mathbf{\beta}}{\mathbf{1 -}\mathbf{tg\ }\mathbf{\alpha \bullet}\mathbf{tg\ }\mathbf{\beta}}.$$

Divu leņķu summas tangenss ir vienāds ar šo leņķu tangensu summas
dalījumu ar skaitļa $1$ un to pašu leņķu tangensu reizinājuma starpību.

*Kā iegūt formulu* $tg(\alpha - \beta)$*?*

> *Analoģiski kā* $tg(\alpha + \beta)$ *vai arī lietojot sakarību*
> $\alpha - \beta = \alpha + ( - \beta)$ *un*
> $tg( - \beta) = - tg\ \beta$*.*

Divu leņķu starpības tangensa formula:

$$tg(\alpha - \beta) = tg\left( \alpha + ( - \beta) \right) = \frac{tg\ \alpha + tg( - \beta)}{1 - tg\ \alpha \bullet tg( - \beta)} = \frac{tg\ \alpha - tg\ \beta}{1 + tg\ \alpha \bullet tg\ \beta}.$$

Ja $\cos{(\alpha - \beta) \neq 0}$, $\cos{\alpha \neq 0}$ un
$\cos{\beta \neq 0}$, tad ir spēkā formula

$$\mathbf{tg}\left( \mathbf{\alpha - \beta} \right)\mathbf{=}\frac{\mathbf{tg\ }\mathbf{\alpha -}\mathbf{tg\ }\mathbf{\beta}}{\mathbf{1 +}\mathbf{tg\ }\mathbf{\alpha \bullet}\mathbf{tg\ }\mathbf{\beta}}.$$

Divu leņķu starpības tangenss ir vienāds ar šo leņķu tangensu starpības
dalījumu ar skaitļa $1$ un šo pašu leņķu tangensu reizinājuma summu.

Aprēķini izteiksmes vērtību!

+--------+---------------------------------------------------------------------------------------------------------------------------------------------+
| **1.** | $$\sin{75{^\circ}} = \sin(45{^\circ} + 30{^\circ}) = \sin{45{^\circ}} \cdot \cos{30{^\circ}} + \sin{30{^\circ}} \cdot \cos{45{^\circ}} =$$  |
|        |                                                                                                                                             |
|        | $$= \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{3}}{2} + \frac{1}{2} \cdot \frac{\sqrt{2}}{2} = \frac{\sqrt{6} + \sqrt{2}}{4}$$                    |
+========+=============================================================================================================================================+
| **2.** | $$\cos{105{^\circ}} = \cos(45{^\circ} + 60{^\circ}) = \cos{45{^\circ}} \cdot \cos{60{^\circ}} - \sin{45{^\circ}} \cdot \sin{60{^\circ}} =$$ |
|        |                                                                                                                                             |
|        | $$= \frac{\sqrt{2}}{2} \cdot \frac{1}{2} - \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{3}}{2} = \frac{\sqrt{2} - \sqrt{6}}{4}$$                    |
+--------+---------------------------------------------------------------------------------------------------------------------------------------------+

Aprēķini izteiksmes $\cos{300{^\circ}}$ vērtību divos dažādos veidos!

*Var lietot redukcijas formulas vai argumentu saskaitīšanas formulas.*

$$\cos{300{^\circ}} = \cos{(360{^\circ} - 60{^\circ})} = \cos{60{^\circ}} = \frac{1}{2}$$

$$\cos{300{^\circ}} = \cos{(360{^\circ} - 60{^\circ})} = \cos{360{^\circ}} \cdot \cos{60{^\circ}} + \sin{360{^\circ}} \cdot \sin{60{^\circ}} = 1 \cdot \frac{1}{2} + 0 \cdot \frac{\sqrt{3}}{2} = \frac{1}{2}$$

*Kādas ir katra risinājuma priekšrocības vai trūkumi?*

Lieto argumentu saskaitīšanas formulas trigonometrisko izteiksmju
vērtību aprēķināšanai un vienkāršošanai (skat. [OL_9_dl_3]{.mark}).

**10., 11. stunda. Divkārša argumenta formulas**

**Sasniedzamie rezultāti**

- Pamato un lieto divkārša argumenta formulas trigonometrisko izteiksmju
  skaitliskās vērtības aprēķināšanai un identiskai pārveidošanai.

- Aprēķina vienas trigonometriskās funkcijas vērtību kā arī divkārša
  argumenta vērtības, ja zināma otras funkcijas vērtība.

*Kā izteiksmi* $\sin{2\alpha}$ *un* $\cos{2\alpha}$ *varētu izteikt ar
leņķa* $\alpha$ *funkcijām?*

Var ievērot, ka $2\alpha = \alpha + \alpha$, un lietot argumentu
saskaitīšanas formulas:

$$\sin{(\alpha + \beta) = \sin{\alpha \bullet \cos{\beta + \cos{\alpha \bullet \sin\beta}}}};$$

$$\cos{(\alpha + \beta) = \cos{\alpha \bullet \cos{\beta - \sin{\alpha \bullet \sin\beta}}}}.$$

Formulās leņķi $\beta$ aizvieto ar $\alpha$ un iegūst:

$$\sin(\alpha + \alpha) = \sin\alpha \bullet \cos{\alpha + \cos{\alpha \bullet \sin\alpha}}\ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \mathbf{\sin}{\mathbf{2}\mathbf{\alpha}\mathbf{=}\mathbf{2}\mathbf{\sin}{\mathbf{\alpha}\mathbf{\bullet}\mathbf{\cos}\mathbf{\alpha}}};$$

$$\cos(\alpha + \alpha) = \cos\alpha \bullet \cos{\alpha - \sin{\alpha \bullet \sin\alpha}}\ \ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \mathbf{\cos}{\mathbf{2}\mathbf{\alpha}}\mathbf{=}\mathbf{\cos}^{\mathbf{2}}\mathbf{\alpha -}\mathbf{\sin}^{\mathbf{2}}\mathbf{\alpha}.$$

Formulē iegūtās formulas vārdiem!

Sinuss no jebkura leņķa ir vienāds ar divkāršotu sinusa un kosinusa
reizinājumu no divas reizes mazāka leņķa.

Kosinuss no jebkura leņķa ir vienāds ar kosinusa kvadrāta no divas
reizes mazāka leņķa un sinusa kvadrāta no tā paša divas reizes mazāka
leņķa starpību.

Uzraksti kā divas reizes mazāka leņķa funkcijas!

  -------------------------------------------------------------------------------
  **1.**   $$\sin{80{^\circ}} = 2\sin{40{^\circ}} \cdot \cos{40{^\circ}}$$
  -------- ----------------------------------------------------------------------
  **2.**   $$\sin{6\alpha} = 2\sin{3\alpha} \bullet \cos{3\alpha}$$

  **3.**   $$\sin{4\beta} = 2\sin{2\beta} \bullet \cos{2\beta}$$

  **4.**   $$\sin\alpha = 2\sin\frac{\alpha}{2} \bullet \cos\frac{\alpha}{2}$$

  **5.**   $$\cos{20{^\circ}} = \cos^{2}{10{^\circ}} - \sin^{2}{10{^\circ}}$$

  **6.**   $$\cos{4\alpha} = \cos^{2}{2\alpha} - \sin^{2}{2\alpha}$$

  **7.**   $$\cos{8\beta} = \cos^{2}{4\beta} - \sin^{2}{4\beta}$$

  **8.**   $$\cos\alpha = \cos^{2}\frac{\alpha}{2} - \sin^{2}\frac{\alpha}{2}$$
  -------------------------------------------------------------------------------

*Kā* $\cos{2\alpha} = \cos^{2}\alpha - \sin^{2}\alpha$ *formulu
uzrakstīt tā, lai vienādības labās puses izteiksme saturētu tikai sinusu
vai tikai kosinusu?*

*Šos pārveidojumus var aplūkot kā risināmus uzdevumus.*

Jālieto formula $\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$, tad
$\cos^{2}{\alpha = 1 - \sin^{2}\alpha}$ un
$\sin^{2}{\alpha = 1 - \cos^{2}\alpha}$.

Tad iegūst:

$$\cos{2\alpha} = \cos^{2}\alpha - \sin^{2}\alpha = (1 - \sin^{2}{\alpha)} - \sin^{2}\alpha = 1 - 2\sin^{2}\alpha;$$

$$\cos^{2}\alpha = \cos^{2}\alpha - \sin^{2}\alpha = \cos^{2}\alpha - (1 - \cos^{2}{\alpha)} = 2\cos^{2}\alpha - 1.$$

*Šīs sakarības var nemācīties no galvas, bet nepieciešamības gadījumā
atcerēties un atkārtot iegūšanas veidu, lietojot formulu*
$\sin^{2}{\alpha + \cos^{2}{\alpha = 1}}$*.*

Aprēķini izteiksmes vērtību!

  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **1.**   $$2\sin{15{^\circ}} \cdot \cos{15{^\circ}} = \sin{30{^\circ}}$$
  -------- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **2.**   $$\sin\frac{\pi}{8} \bullet \cos\frac{\pi}{8} = \frac{1}{2} \cdot 2\sin\frac{\pi}{8} \bullet \cos\frac{\pi}{8} = \frac{1}{2}\sin\frac{\pi}{4} = \frac{1}{2} \cdot \frac{\sqrt{2}}{2} = \frac{\sqrt{2}}{4}$$

  **3.**   $$\cos^{2}{15{^\circ}} - \sin^{2}{15{^\circ}} = \cos{30{^\circ}} = \frac{\sqrt{3}}{2}$$

  **4.**   $$\sin^{2}\frac{\pi}{8} - \cos^{2}\frac{\pi}{8} = - \cos\frac{\pi}{4} = - \frac{\sqrt{2}}{2}$$

  **5.**   $$1 - 2\sin^{2}{75{^\circ}} = \sin{150{^\circ}} = \sin(180{^\circ} - 30{^\circ}) = \sin{30{^\circ}} = \frac{1}{2}$$

  **6.**   $$2\cos^{2}{105{^\circ}} - 1 = \cos{210{^\circ}} = \cos(180{^\circ} + 30{^\circ}) = - \cos{30{^\circ}} = - \frac{\sqrt{3}}{2}$$
  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lieto divkārša argumenta formulas trigonometrisko izteiksmju vērtību
aprēķināšanai un vienkāršošanai (skat. [OL_9_dl_4]{.mark}).

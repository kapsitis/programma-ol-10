**10. Trigonometriskie vienādojumi** (16 stundas)

Metodiskais komentārs par tematu

Tematā uzsvars ir uz trigonometrisko pamatvienādojumu risināšanu
$\sin(ax + b) = c$,\
$\cos(ax + b) = c$, $tg(ax + b) = c$ un jau iepriekš apgūto vienādojumu
risināšanas metožu (sadalīšana reizinātājos, substitūcija) lietošanu.

Trigonometrisko vienādojumu atrisinājumu noteikšanu jāsaista arī ar
atbilstošās funkcijas grafika krustpunktiem ar taisni $y = c$. Lai
noteiktu vienādojuma atrisinājumu dotajā intervālā, vispirms uzraksta
vienādojuma vispārīgo vienādojumu un pēc tam nosaka, kuras saknes
atrodas dotajā intervālā.

Lai apgūtu trigonometrisko vienādojumu risināšanas metodes, skolēniem
jābūt zināšanām par trigonometriskām funkcijām un prasmēm pārveidot
trigonometriskas izteiksmes un risināt kvadrātvienādojumus.

**!!!** Iekļauti arī vienādojumi ar tangensu, kas nav optimālā līmeņa
saturs, tāpēc tos skolotājs var nemācīt un attiecīgos uzdevumus
nerisināt.

Materiālu apraksts

Metodiskā materiāla sākumā ir dots temata sadalījums pa stundām vai
stundu grupām, attiecīgi norādot stundas tematu un skolēnam sasniedzamos
rezultātus.

Katrai stundai vai stundu grupai dots svarīgākais par nepieciešamo
teoriju, norāde uz darba lapām vai uzdevumu atrisinājumu paraugi,
ilustrācijas. Teksts, kas ir *slīprakstā pelēkā krāsā*, ir ieteikumi
skolotājam, kādus jautājumus uzdot skolēniem, kam pievērst uzmanību,
dažas iespējas darba organizēšanai. Par piemērotāko metodi lemj
skolotājs atbilstoši skolēnu vajadzībām un iespējām (dažādi metodiskie
ieteikumi darba organizācijai un metodēm ir pieejami *Skola2030*
izstrādātajā Matemātika I programmas paraugā). Saturs, kas pārsniedz
standarta sasniedzamos rezultātus, ir pelēkā rāmī un apzīmēts Nav
standarta SR, to skolotājs var dot apgūt papildus, īpaši tiem skolēniem,
kas plāno studēt STEM jomā.

Materiālā ir norādes uz darba lapām, ko skolotājs var dot skolēniem
drukātā formātā vai arī elektroniski kā pdf, ja skolēni pierakstiem
lieto planšetes. Darba lapas var pārveidot par uzdevumu lapām, izdzēšot
risināšanai paredzēto vietu, kā arī uzdevumus var salikt prezentācijas
slaidos un risināt uz interaktīvās tāfeles. Darba lapās iekļauts vairāk

Visām darba lapām ir doti atrisinājumi.

Materiālā ir piedāvāti arī daži rakstiski formatīvās vērtēšanas darbi,
kas iekļauj vairākus sasniedzamos rezultātus, doti arī atrisinājumi un

vajadzībām.

Tematā ir paredzēta mācību stunda temata kopsavilkumam, tiek piedāvāta
uzdevumu lapa ar dažādiem uzdevumiem. Skolotājs var pielāgot gatavošanos
temata nobeiguma darbam, šādu stundu var arī neiekļaut plānā, bet
papildu stundu veltīt kāda sasniedzamā rezultāta nostiprināšanai vai
temata padziļinājumam.

Temata nobeiguma darbam piedāvāti divi līdzvērtīgi varianti ar
atrisinājumiem un vērtēšanas kritērijiem. Noslēguma pārbaudes darbi

kārtības nolikumam.

**1.--5. stunda. Trigonometriskie pamatvienādojumi**

**Sasniedzamie rezultāti**

- Spriež par vienādojuma atrisinājumu noteiktā intervālā gan grafiski,
  gan no vispārīgā atrisinājuma.

- Atrisina trigonometriskos pamatvienādojumus $\sin{x = a}$,
  $\cos{x = a}$ un $tg\ x = a$, lietojot atrisinājumu trigonometrisko
  vienības riņķi.

- Atrisina trigonometriskos vienādojumus, kas pārveidojami formā
  $\sin{(ax + b)} = c$,\
  $\cos{(ax + b)} = c$ un $tg(ax + b) = c$, lietojot trigonometrisko
  vienības riņķi.

*Vai dažreiz vienādojuma atrisinājumu var noteikt, lietojot atbilstošo
funkciju grafikus un nosakot to krustpunktu abscisas vērtību?*

> *Jā, to jau dažos gadījumos darīja iepriekšējā tematā par
> trigonometriskām funkcijām. Piemēram, ja funkcijas* $y = \sin x$
> *vērtība ir* $\frac{\pi}{6}$*, nosaki atbilstošo argumenta vērtību.*

*Kā rīkoties, ja funkcija būtu*
$y = \sin\left( \frac{\pi}{10} + 7x \right)$*?*

> *Nonāk pie secinājuma, ka funkciju precīzi uzzīmēt ir grūti un pēc tam
> no grafika var tikai aptuveni noteikt atbilstošās* $x$ *vērtības.
> Nepieciešama metode, kā risināt šāda veida vienādojumus analītiski.*

Vienādojumu ar vienu nezināmo sauc par trigonometrisku vienādojumu, ja
nezināmais tajā ietilpst tikai kā trigonometriskās funkcijas arguments.

Piemēri

  ------------------------------------------------------------------------------------------------------------------
  $$\cos{2x} = 0,5$$            $${tg}\left( \frac{\pi}{4} - x \right) = \sqrt{3}$$   $${tg}x = {tg}^{2}x - 6$$
  ----------------------------- ----------------------------------------------------- ------------------------------
  $$\cos^{2}x - 2\sin x = 0$$   $$2\sin x\cos x = 5$$                                 $$\sin{3x} - 3\cos{6x} = 2$$

  ------------------------------------------------------------------------------------------------------------------

Vispirms aplūko trigonometriskos pamatvienādojumus.

  --------------------------------------------------------------
  $$\sin{x = a}$$      $$\cos{x = a}$$      $${tg}x = a$$
  -------------------- -------------------- --------------------

  --------------------------------------------------------------

*Vienādojumu* ${ctg}x = a$ *neaplūko, jo to var pārveidot par
vienādojumu* ${tg}x = \frac{1}{a}$.

Trigonometriskos pamatvienādojumus var risināt, lietojot trigonometrisko
vienības riņķi (TVR).

*Tā kā Matemātika I kursā nav paredzēts apgūt ciklometriskās funkcijas
un to īpašības, tad nevar lietot vispārīgās formulas trigonometrisko
pamatvienādojumu atrisināšanai.*

**Fizika (svārstības)**

Ķermenis svārstās pēc likuma
$h(t) = 2\sin\left( \frac{\pi}{3}t \right)$, kur $h$ ir novirze metros,
$t$ ir laiks sekundēs. Aprēķini laiku, kad ķermenis pirmo reizi sasniedz
1 metra augstumu!

*Situācijas analīze, problēmas izpratne.*

> Jāaprēķina laiks $t$, kad novirze ir 1 metrs.

*Matemātiskā modeļa izveide, uzraksta vienādojumu.*

> $$h(t) = 1\ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ 2\sin\left( \frac{\pi}{3}t \right) = 1$$

*Atrisina vienādojumu.*

> $$2\sin\left( \frac{\pi}{3}t \right) = 1\ \ \ \ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \ \sin\left( \frac{\pi}{3}t \right) = \frac{1}{2}$$
>
> $$\frac{\pi}{3}t = \left\lbrack \ \begin{array}{r}
> \frac{\pi}{6} + 2\pi n \\
> \frac{5\pi}{6} + 2\pi n
> \end{array} \right.\ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \ t = \left\lbrack \ \begin{array}{r}
> \frac{1}{2} + 6n \\
> \frac{5}{2} + 6n
> \end{array} \right.\ ,\ \ n\mathbb{\in Z}$$

*Rezultāta interpretācija kontekstā.*

> Ķermenis pirmo reizi sasniedz $1\ $m augstumu, ja $n = 0$, tātad
> $t = \frac{1}{2}$.

Atbilde. Ķermenis pirmo reizi sasniedz 1 m augstumu pēc 0,5 sekundēm.

**Ģeogrāfija (dienas garums)**

Dienas garumu stundās noteiktā vietā apraksta modelis:
$D(t) = 12 + 4\sin\left( \frac{\pi}{6}(t - 3) \right)$, kur $t$ ir
mēneša kārtas numurs. Aprēķini, kurā mēnesī dienas garums ir 14 stundas!

*Situācijas analīze, problēmas izpratne.*

> Jāaprēķina mēneša kārtas numurs $t$, kad dienas garums ir 14 stundas.

*Matemātiskā modeļa izveide, uzraksta vienādojumu.*

> $$D(t) = 14\ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ 12 + 4\sin\left( \frac{\pi}{6}(t - 3) \right) = 14$$

*Atrisina vienādojumu.*

> $$4\sin\left( \frac{\pi}{6}(t - 3) \right) = 2\ \ \ \ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \ \sin\left( \frac{\pi}{6}(t - 3) \right) = \frac{1}{2}$$
>
> $$\frac{\pi}{6}t - \frac{\pi}{2} = \left\lbrack \begin{array}{r}
> \frac{\pi}{6} + 2\pi n \\
> \frac{5\pi}{6} + 2\pi n
> \end{array} \right.\ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \frac{\pi}{6}t = \left\lbrack \ \begin{array}{r}
> \frac{2\pi}{3} + 2\pi n \\
> \frac{4\pi}{3} + 2\pi n
> \end{array} \right.\ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ t = \left\lbrack \ \begin{array}{r}
> 4 + 12n \\
> 8 + 12n
> \end{array} \right.\ ,\ \ n\mathbb{\in Z}$$

*Rezultāta interpretācija kontekstā.*

> $$t \in \left\{ 1;2;\ldots;12 \right\}\ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ t_{1} = 4;\ \ t_{2} = 8$$

Atbilde. Dienas garums 14 stundas ir aprīlī (4. mēnesī) un augustā (8.
mēnesī).

**Fizika (rotējošs ritenis)**

Punkta augstumu uz rotējoša riteņa apraksta funkcija
$h(t) = 1,5 + 1,5\cos{\pi t}$, kur $h$ ir augstums metros un $t$ ir
sekundēs. Aprēķini laiku, kad punkts atrodas 3 metru augstumā!

*Situācijas analīze, problēmas izpratne.*

> Jāaprēķina laiks $t$, kad punkts atrodas 3 metru augstumā.

*Matemātiskā modeļa izveide, uzraksta vienādojumu.*

> $$h(t) = 3\ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ 1,5 + 1,5\cos{(\pi t)} = 3$$

*Atrisina vienādojumu.*

> $$1,5\cos{\pi t} = 1,5\ \ \ \ \ \ \ \ \ \ \ \ \  \Rightarrow \ \ \ \ \ \ \ \ \ \ \ \cos{\pi t} = 1$$
>
> $$\pi t = 0 + 2\pi n$$
>
> $t = 2n$, kur $n\mathbb{\in Z}$ un $t \geq 0$.

*Rezultāta interpretācija kontekstā.*

> Process ir periodisks, tāpēc $t = 0;2;4;6;\ldots$ sekundes.

Atbilde. Punkts atrodas 3 metru augstumā ik pēc 2 sekundēm (t. i.,
$t = 0;\ 2;\ 4;\ 6;\ldots$ sekundes).

**Sasniedzamie rezultāti**

- Pārbauda zināšanas un prasmes par trigonometriskiem vienādojumiem un

uzdevumu skaitu,\

nolikumu.

Ieteicama noslēguma pārbaudes darba kļūdu analīze. Skolotājs to organizē

| **(2)** | $m^3-n^3=$ | $\color{blue}{(m-n)(m^2+mn+n^2)}$ |

| **(5)** | $t^3+27=$ | $\color{blue}{(t+3)(t^2-3t+9)}$ |
| **(6)** | $125-x^3=$ | $\color{blue}{(5-x)(25+5x+x^2)}$ |

| **(8)** | $a^3b^3-1=$ | $\color{blue}{(ab-1)(a^2b^2+ab+1)}$ |
| **(9)** | $8a^3-\dfrac{1}{125}=$ | $\color{blue}{\left(2a-\dfrac{1}{5}\right)\left(4a^2+\dfrac{2}{5}a+\dfrac{1}{25}\right)}$ |
| **(10)** | $1+216a^3=$ | $\color{blue}{(1+6a)(1-6a+36a^2)}$ |
| **(11)** | $343y^3+27x^3=$ | $\color{blue}{(7y+3x)(49y^2-21xy+9x^2)}$ |
| **(12)** | $2+2a^3=$ | $\color{blue}{2(1+a^3)=2(1+a)(1-a+a^2)}$ |
| **(13)** | $40a^3-5=$ | $\color{blue}{5(8a^3-1)=5(2a-1)(4a^2+2a+1)}$ |
| **(14)** | $a^6b^9+64t^{12}=$ | $\color{blue}{(a^2b^3+4t^4)(a^4b^6-4a^2b^3t^4+16t^8)}$ |

**FVD 1 -- vienādojumu sistēmu risināšana ar ievietošanas un
saskaitīšanas paņēmienu**

**1.** (5 p.) Atrisini vienādojumu sistēmu, lietojot ievietošanas
paņēmienu!

$$\left\{ \ \begin{array}{r}
x + 2y = 7 \\
xy = 5\ \ \ \ \ \ \ \ 
\end{array} \right.\ $$

**2.** (5 p.) Atrisini vienādojumu sistēmu, lietojot saskaitīšanas
paņēmienu!

$$\left\{ \ \begin{array}{r}
2y - 3x^{2} = 1 \\
{3x}^{2} + y = 5\ \ 
\end{array} \right.\ $$

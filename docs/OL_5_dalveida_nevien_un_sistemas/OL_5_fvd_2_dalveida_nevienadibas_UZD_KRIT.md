**FVD 2 -- daļveida nevienādība**

**1.** (2 p.) Uzraksti vienu pozitīvu skaitli, kurš pieder nevienādības
$\frac{x + 6}{4 - x} > 0$ atrisinājuma kopai!

**2.** (1 p.) Nosaki, kuras nevienādības atrisinājumam atbilst dotais
zīmējums!

![Attēls, kurā ir rinda, diagramma Mākslīgā intelekta ģenerēts saturs
var būt nepareizs.](media/image1.png){width="2.1336056430446195in"
height="0.8267716535433071in"}

  A   $$\frac{x + 4}{1 - x} > 0$$   B   $$\frac{x + 4}{1 - x} \leq 0$$   C   $$\frac{x - 1}{x + 4} \leq 0$$   D   $$\frac{x + 4}{x - 1} < 0$$

**3.** (5 p.) Atrisini nevienādību!

$$\frac{x^{2} - 3x}{2x + 4} \leq 0$$

**4.** (7 p.) Atrisini nevienādību sistēmu!

$$\left\{ \ \begin{array}{r}
 - 2(x - 6) \geq x - 3 \\
\frac{x + 5}{6 - x - x^{2}} < 0\ \ \ \ \ \ \ 
\end{array} \right.\ $$

Darbs plānots 15--20 min, lai pārbaudītu, vai skolēns nosaka konkrētas
vērtības piederību atrisinājuma kopai, nosaka nevienādību, ja dots
atbilstošs grafiskais attēlojums, kā arī atrisina daļveida nevienādību
un nevienādību sistēmu.

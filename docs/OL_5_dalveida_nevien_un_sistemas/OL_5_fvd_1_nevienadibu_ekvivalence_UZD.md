**FVD 1 -- nevienādības un to ekvivalence**

**1.** (2 p.) Nosaki un pamato, vai skaitlis $3$ ir nevienādības
$\frac{x^{2} - 4}{x - 1} > 0$ atrisinājums!

**2.** (1 p.) Nevienādība $2x - 7 < 5x + 3$ ir ekvivalenta ar
nevienādību

A $\ \ \ \ 7x - 7 < 3$ B $\ \ \  - 3x < 10$ C $\ \ \ \  - 3x < 4$

**3.** (8 p.) Atrisini nevienādību!

  -----------------------------------------------------------------------
  3.1. (3 p.) $$\frac{4}{2x - 5} \leq 0$$
  ----------- -----------------------------------------------------------
  3.2. (5 p.) $$\frac{x + 6}{4x - 12} \geq 0$$

  -----------------------------------------------------------------------

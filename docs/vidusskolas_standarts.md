# Vidusskolas standarts

Sk. [Ministru kabineta noteikumi Nr. 416](https://likumi.lv/ta/id/309597-noteikumi-par-valsts-visparejas-videjas-izglitibas-standartu-un-visparejas-videjas-izglitibas-programmu-paraugiem#n-703526)

Plānotie skolēnam sasniedzamie rezultāti vispārējā vidējā izglītības standartā, beidzot vispārīgā, optimālā vai augstākajā apguves līmenī.

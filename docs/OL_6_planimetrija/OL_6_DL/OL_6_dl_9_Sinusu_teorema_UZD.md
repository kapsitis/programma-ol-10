1.  Trijstūrī $ABC$ $\sphericalangle B = 30{^\circ}$,
    $\sphericalangle C = 45{^\circ}$ un $AC = 8$. Aprēķini $AB$!

2.  Trijstūrī $ABC$ $\sphericalangle B = 60{^\circ}$,
    $\sphericalangle A = 45{^\circ}$ un $BC = 12$. Aprēķini $AC$!

3.  Trijstūrī $ABC$ $\sphericalangle C = 105{^\circ}$,
    $\sphericalangle A = 30{^\circ}$ un $AC = 5$. Aprēķini $BC$!

4.  Trijstūrī $ABC$ $\sphericalangle A = 30{^\circ}$,
    $AB = \frac{\sqrt{2}}{2}$ un $BC = \frac{1}{2}$. Aprēķini
    $\sphericalangle C$!

5.  Trijstūrī $ABC$ mala $AB = \frac{1}{2}$,
    $\sphericalangle A = 60{^\circ}$ un $BC = \frac{1}{2}$. Aprēķini
    $\sphericalangle C$!

6.  Trijstūrī $MNK$ $\sphericalangle M = 30{^\circ}$, $\sin N = 0,4$ un
    $MK = 8$. Aprēķini $NK$!

7.  Trijstūrī $ABC$ $\sin A = 0,9$, $\sin C = 0,8$ un $AB = 4$. Aprēķini
    $BC$!

8.  Trijstūrī $ABC$ $\sphericalangle A = 62{^\circ}$,
    $\sphericalangle C = 93{^\circ}$ un $AC = 2$. Aprēķini gan malu $BC$
    un $AB$ precīzās vērtības, gan aptuvenās ar precizitāti līdz
    simtdaļām!

9.  Paralelograma diagonāle ar malām veido $45{^\circ}$ un $30{^\circ}$
    leņķus, bet mala pret $30{^\circ}$ leņķi ir $\sqrt{2}$ gara.
    Aprēķini paralelograma otras malas garumu!

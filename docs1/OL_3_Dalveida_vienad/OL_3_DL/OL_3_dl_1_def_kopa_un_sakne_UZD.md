**2**. Nosaki un ar aprēķiniem pamato, vai dotais skaitlis ir

  1\.   $$\frac{45}{x^{2}} = 5;\ \ \ x = - 3$$                                     1\.   $$\frac{36}{x^{2}} = 9;\ \ \ x = 2$$
  ----- -------------------------------------------------------------------------- ----- ---------------------------------------------------------------------------
  2\.   $$\frac{x - 1}{x} = - 1;\ \ \ x = 0$$                                      2\.   $$\frac{x + 1}{x - 2} = 5;\ \ \ x = 2$$

  3\.   $$\frac{5x + 1}{x + 1} = \frac{x + 2}{x};\ \ \ x = - \frac{1}{2}$$         3\.   $$\frac{3x + 1}{x + 1} = \frac{x + 4}{x};\ \ \ x = - 1$$

  4\.   $$\frac{8}{x} = 3x + 2;\ \ \ \ \ \ x_{1} = - 2,\ \ x_{2} = \frac{4}{3}$$   4\.   $$\frac{3}{x^{2} + 2} = \frac{1}{x};\ \ \ \ \ \ x_{1} = 1,\ \ x_{2} = 2$$

  5\.   $$\frac{6x + 12}{x^{2} + x - 2} = x;\ \ \ x = 3$$                          5\.   $$\frac{3x - 9}{x^{2} - 2x - 3} = 1;\ \ \ x = 2$$

  6\.   $$\frac{x - 4}{x - 5} + \frac{x - 6}{x + 5} = 2;\ \ \ x = 1$$              6\.   $$\frac{5}{x - 2} - \frac{4}{x - 3} = \frac{1}{x};\ \ \ x = - 3$$

  7\.   $$\frac{x^{3} - 8}{x - 2} = 24x - 36;\ \ \ x = 20$$                        7\.   $$\frac{x^{3} - 125}{x - 5} = 8x + 35;\ \ \ x = 2$$
